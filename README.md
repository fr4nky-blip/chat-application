# Chat Application (Frontend Prototype)

This is a simple chat application UI built using HTML, CSS, and JavaScript.

## Features
- Login & Register UI
- Chat interface layout
- User list UI
- Message sending simulation (frontend only)

## Tech Stack
- HTML
- CSS
- JavaScript

## Current Status
- Backend (DB, authentication, real-time messaging) is NOT connected.
- This project focuses on frontend structure and UI flow.
- Backend integration using Spring Boot + WebSocket is planned.

## Future Improvements
- Connect login to database
- Real-time messaging using WebSocket
- User authentication
- Message persistence
