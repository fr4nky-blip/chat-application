# OnyxApp

OnyxApp is a responsive chat application built with HTML, CSS, and JavaScript.

## Features
- Login & Registration
- User list and chat selection
- Real-time message simulation
- Clean, intuitive interface

## Tech Stack
- HTML
- CSS
- JavaScript

## Deployment
- Deployed on Vercel / GitHub Pages
- Fully functional UI accessible in browser
